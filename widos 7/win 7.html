Windows 7

A Microsoft egyik legnépszerűbb operációs rendszere, amely 2009 októberében jelent meg. A Windows Vista utódjaként a Windows 7 jelentős fejlődést hozott a felhasználói élmény, a stabilitás és a teljesítmény terén.

Az "Aero" felület, az új taskbar, a javított fájlkezelő és számos egyéb innováció tette a Windows 7-et a számítástechnika történetének egyik legkedveltebb operációs rendszerévé.

További információ Windows 7 élmény

A Windows 7 Története

Egy Sikerstory

A Windows 7 fejlesztése a Windows Vista megjelenése után kezdődött, amely számos kritikát kapott a teljesítménye és kompatibilitási problémái miatt. A Microsoft célja egy stabilabb, gyorsabb és felhasználóbarátabb operációs rendszer készítése volt.

A Windows 7 hivatalosan 2009. október 22-én jelent meg, és hamarosan a világ legnépszerűbb operációs rendszerévé vált. 2012 októberében már 670 millió licencelt példány volt forgalomban világszerte.

2020 januárjában ért véget a hivatalos támogatás, de még ma is milliók használják szerte a világon.

Főbb Jellemzők

Aero Felület

Átlátszó ablakkeretek, live thumbnails, Aero Peek és Aero Shake funkciók, amelyek forradalmasították a windows kezelését.

Új Taskbar

A Superbar elnevezésű új taskbar lehetővé tette az alkalmazások rögzítését, a jump list-eket és az egyszerűbb ablakkezelést.

Snap Funkció

Az ablakok egyszerű átméretezése és pozicionálása az egér segítségével, ami jelentősen javította a multitasking élményt.

Windows 7 Verziók

Windows 7 Starter

Alapverzió, főleg netbookok számára. Korlátozott funkcionalitás, nincs Aero felület, nem támogatja a 64 bites architektúrát.

Windows 7 Home Basic

Fejlettebb, mint a Starter, de még mindig korlátozott. Főleg fejlődő országokban forgalmazták.

Windows 7 Home Premium

A legnépszerűbb otthoni verzió. Teljes Aero felület, Media Center, multimédiás funkciók.

Windows 7 Professional

Kisvállalati verzió. Tartalmazza a Windows XP módot, hálózati biztonsági funkciók, tartománycsatlakozás.

Windows 7 Ultimate

Legmagasabb szintű verzió. Minden funkció megtalálható, BitLocker titkosítás, több nyelv támogatása.

Aero Peek

Az Aero Peek funkció lehetővé tette, hogy az egérkurzort a taskbar ikonjaira vivve előnézetet kapjunk az ablak tartalmáról.

Snap & Shake

Az ablakok egyszerű átméretezése képernyő széléhez húzással (Snap) és felesleges ablakok eltüntetése rázkódtatással (Shake).

Jump Lists

A jobbklikk a taskbar ikonokon megnyitotta a Jump List-et, gyors hozzáférést biztosítva a gyakran használt fájlokhoz és funkciókhoz.
